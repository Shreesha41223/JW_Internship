# 📘 Django Dictionary App (Using NLTK)

A simple Django app to get **meanings**, **synonyms**, and **antonyms** of any word using the NLTK WordNet library.

---

## 🔍 Features

- Get definitions
- Find synonyms and antonyms
- Clean, simple UI
